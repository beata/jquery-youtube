jquery-youtube
==============

A jQuery plugin that inserts a iframe from youtube video url.

```html
<div class="js-youtube" data-url="http://www.youtube.com/watch?v=DDs5bXh4erM"></div>
```

## Options

* `url` (Required)  
    The youtube url of the video.

* `width` (Optional)  
    Default: 300  
    Width of the iframe

* `height` (Optional)  
    Default: 300 * 0.75  
    Height of the iframe
